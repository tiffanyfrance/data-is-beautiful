There are three main ways in which risk of fatality of a certain mode of travel can be measured: Deaths per billion typical journeys taken, deaths per billion hours traveled, or deaths per billion kilometers traveled. This chart visualizes these statistics for the United Kingdom from 1990â€“2000.

The graph is laid out using a log scale which results in downplaying the high casuality rate for motorcycle drivers, and was choosen because without it, the rest of the data would not be comparable.

Data source can be found on wikipedia. The original source also includes Paragliding, Skydiving, and Space Shuttle, which were too rare or insignificant to be included in this analysis.
